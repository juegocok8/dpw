🐬 PyMySQL conector en Python puro

El puente que permite que tu código Python hable con MySQL / MariaDB: guardar, consultar, actualizar y borrar datos.

💡 ¿Qué es PyMySQL en términos simples?
Es una librería de Python que funciona como un cliente o conector para interactuar con bases de datos MySQL y MariaDB. Actúa como un intérprete que traduce las órdenes de Python a consultas SQL y viceversa.

✨ ¿Qué lo hace especial?

A diferencia de otros conectores como mysql-connector-python, PyMySQL destaca por ser 100% Python puro. Esto trae ventajas clave:

🧩 Sin dependencias externas No necesitas instalar bibliotecas de C en tu sistema ni lidiar con configuraciones complejas de compilación. Basta con un pip install PyMySQL.
⚡ Compatible con PyPy Funciona perfectamente con PyPy, ideal si buscas un rendimiento optimizado en entornos que no usan CPython (el intérprete estándar).
🔄 Soporte moderno Compatible con la mayoría de versiones de MySQL (5.x, 8.x) y MariaDB, manteniéndose actualizado.

🔁 ¿Cómo funciona el flujo típico?

Usar PyMySQL sigue un patrón bastante claro y universal. Estos son los pasos fundamentales:

  1. 📡 Establecer conexión — Proporcionas el host, usuario, contraseña, puerto y nombre de la base de datos.
  2. 🎯 Crear un Cursor — El cursor es el objeto que te permite ejecutar sentencias SQL y recorrer los resultados.
  3. 📝 Ejecutar SQL — Envías comandos como SELECT, INSERT, UPDATE o DELETE.
  4. 💾 Hacer Commit — Si cambias datos (INSERT, UPDATE, DELETE), debes confirmar la acción usando commit().
  5. 🔌 Cerrar — Siempre es buena práctica cerrar el cursor y la conexión para liberar recursos.

🧪 Ejemplo mínimo de código

import pymysql

# 1. Establecer conexión
conexion = pymysql.connect(
    host='localhost',
    user='root',
    password='tu_contraseña',
    database='mi_bd',
    port=3306
)

try:
    # 2. Crear cursor
    cursor = conexion.cursor()

    # 3. Ejecutar SQL
    sql = "SELECT nombre, email FROM USERS WHERE rol = 'admin'"
    cursor.execute(sql)

    # Obtener resultados
    for (nombre, email) in cursor:
        print(f"Admin: {nombre} - {email}")

    # Ejemplo de INSERT con commit
    insert_sql = "INSERT INTO CATEGORIES (name, created_at) VALUES ('Trabajo', NOW())"
    cursor.execute(insert_sql)
    # 4. Confirmar cambios
    conexion.commit()
    print("✅ Categoría agregada correctamente")

finally:
    # 5. Cerrar recursos
    cursor.close()
    conexion.close()
🚀 Integración con Django
PyMySQL es clave en proyectos Django con MySQL: solo necesitas agregar import pymysql; pymysql.install_as_MySQLdb() en el __init__.py de tu proyecto y configurar el motor django.db.backends.mysql. Sin dolores de cabeza con compilaciones nativas.

🧠 ¿Por qué elegir PyMySQL en tu próximo proyecto?

  • Simplicidad radical: no necesitas compilar drivers extraños.
  • Comunidad activa: es mantenido y se actualiza con frecuencia.
  • Perfecto para frameworks: se integra sin fricción con Django, Flask, FastAPI (usando SQLAlchemy o directo).
  • Modo de depuración claro: al ser Python puro, los errores son más legibles.
🧑‍💻 Dato final
Si ya usas MySQL/MariaDB y quieres un conector que no complique tu entorno de desarrollo ni producción, PyMySQL es la opción más amigable y confiable. Además, es la base perfecta para entender cómo Python interactúa con motores SQL sin abstracciones innecesarias.
