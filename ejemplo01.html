Plantilla HTML5 + CSS Responsivo

El siguiente código muestra una estructura de 3 columnas con cabecera y pie de página.

index.html (Vista de Código) 

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Creación de plantilla</title>

    <style>

        /* ==================================================
            CONFIGURACIÓN GENERAL
            ================================================== */

        /* Quitamos el margen por defecto que trae el navegador */
        body {
            margin: 0;
            font-family: Arial, Helvetica, sans-serif;
        }

        /* La propiedad box-sizing permite que el padding
            no aumente el tamaño real del elemento */
        * {
            box-sizing: border-box;
        }

        /* ==================================================
            ESTILO DEL ENCABEZADO
            ================================================== */

        /* Creamos el diseño visual del header */
        .header {
            background-color: #f1f1f1;   /* Color de fondo */
            padding: 20px;               /* Espaciado interno */
            text-align: center;          /* Centrar texto */
        }

        /* ==================================================
            ESTILO DEL MENÚ DE NAVEGACIÓN
            ================================================== */

        /* Contenedor principal del menú */
        .topnav {
            overflow: hidden;           /* Evita que los elementos flotantes se salgan */
            background-color: #333;     /* Color de fondo oscuro */
        }

        /* Estilo de los enlaces dentro del menú */
        .topnav a {
            float: left;                /* Coloca los enlaces uno al lado del otro */
            display: block;             /* Permite aplicar padding */
            color: #f2f2f2;             /* Color del texto */
            text-align: center;         /* Centrar texto */
            padding: 14px 16px;         /* Espaciado interno */
            text-decoration: none;      /* Quitar subrayado */
        }

        /* Efecto cuando el mouse pasa sobre el enlace */
        .topnav a:hover {
            background-color: #ddd;     /* Cambia el fondo */
            color: black;               /* Cambia el color del texto */
        }

        /* ==================================================
            DISEÑO DE LAS COLUMNAS
            ================================================== */

        /* Clase base para todas las columnas */
        .column {
            float: left;    /* Hace que las columnas se coloquen horizontalmente */
            padding: 10px;  /* Espaciado interno */
        }

        /* Columnas laterales (izquierda y derecha) */
        .column.side {
            width: 15%;     /* Ancho del 15% */
            background-color: #f9f9f9;
        }

        /* Columna central (contenido principal) */
        .column.middle {
            width: 70%;     /* Ancho del 70% */
        }

        /* Limpiar el efecto float para que el footer
            no se suba al lado de las columnas */
        .row:after {
            content: "";
            display: table;
            clear: both;
        }

        /* ==================================================
            DISEÑO RESPONSIVO
            ================================================== */

        /* Cuando la pantalla sea menor a 600px,
            las columnas ocuparán el 100% del ancho
            y se mostrarán una debajo de otra */
        @media screen and (max-width: 600px) {
            .column {
                width: 100%;
            }
        }

        /* ==================================================
            PIE DE PÁGINA
            ================================================== */

        .footer {
            background-color: #f1f1f1;  /* Color de fondo */
            padding: 10px;              /* Espaciado interno */
            text-align: center;         /* Centrar texto */
        }

    </style>
</head>
